Lesson 1: Basic HTML Setup

មេរៀនទី ១៖ ការរៀបចំ HTML មូលដ្ឋាន

Objectives:

គោលបំណង:

  • Understand basic HTML structure
  • Create your first HTML file
  • Use tags like <!DOCTYPE>, <html>, <head>, and <body>
  • យល់ដឹងអំពីរចនាសម្ព័ន្ធ HTML
  • បង្កើតឯកសារ HTML ដំបូងរបស់អ្នក
  • ប្រើស្លាកដូចជា <!DOCTYPE>, <html>, <head>, និង <body>

Step 1 ✅

ជំហានទី ១ ✅

HTML documents start with <!DOCTYPE html> and contain <html>, <head>, and <body> tags to structure the page.

ឯកសារ HTML ត្រូវចាប់ផ្តើមដោយ <!DOCTYPE html> ហើយមាន <html>, <head> និង <body> ដើម្បីរៀបចំទំព័រ។

👉 Your Task: Type <!DOCTYPE html> at the very top to complete the skeleton.

👉 ភារកិច្ច: សរសេរ <!DOCTYPE html> នៅលើគេក្នុងកូដដើម្បីបញ្ចប់រចនាសម្ព័ន្ធ។

<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello Cambodia!</h1>
  </body>
</html>
