Step 1: Start with the HTML Skeleton

ឆុមំទី ឧើឆ្ភាសាខេជាមុករាជណុក HTML

Every HTML document starts with <!DOCTYPE html> and uses <html> as the root. Inside it, we include a <head> and a <body>. Let’s build the skeleton of your first webpage!

HTML រាលអេកសាដទុនច្រូងឆេនព្រ <!DOCTYPE html> អាច្រូ <html> ឆ្រុងឆាងបុរកុក្ពងព្រកក្នាងលៅ <head> ច្រាក <body> កាំំិសុង់

🎉 You did it! Keep going!
