Step 7 ✅

ជំហានទី ៧ ✅

In HTML, you can create lists using <ul> for unordered lists and <ol> for ordered lists. Each item inside the list should be wrapped in a <li> tag.

នៅក្នុង HTML អ្នកអាចបង្កើតបញ្ជីដោយប្រើ <ul> សម្រាប់ unordered list និង <ol> សម្រាប់ ordered list។ រាល់ធាតុត្រូវសរសេរជាមួយ tag <li>។

👉 Your Task: Type <ul><li>Apple</li></ul> in the box below.

👉 ភារកិច្ច: វាយ <ul><li>Apple</li></ul> ខាងក្រោម។
