Lesson 7: Lists in HTML

មេរៀនទី ៧៖ បញ្ជីក្នុង HTML

Objectives:

គោលបំណង:

  • Learn how to use <ul>, <ol>, and <li> tags
  • Understand the difference between ordered and unordered lists
  • Practice writing your own list
  • សិក្សាពីការប្រើស្លាក <ul>, <ol>, និង <li>
  • យល់ពីភាពខុសគ្នារវាងបញ្ជីមានលំដាប់ និងគ្មានលំដាប់
  • អនុវត្តន៍សរសេរបញ្ជីផ្ទាល់ខ្លួន

Step 7 ✅

ជំហានទី ៧ ✅

Use <ul> or <ol> for lists. Each list item goes inside an <li> tag.

ប្រើ <ul> ឬ <ol> សម្រាប់បញ្ជី។ ធាតុក្នុងបញ្ជីគ្រប់ធាតុគួរត្រូវសរសេរនៅក្នុង <li>។
