Lesson 10: HTML Comments

មេរៀនទី ១០៖ កំណត់ចំណាំក្នុង HTML

Objectives:

គោលបំណង:

  • Understand the purpose of HTML comments
  • Learn how to write a comment in your code
  • Practice using a comment tag in HTML
  • យល់ពីគោលបំណងនៃកំណត់ចំណាំក្នុង HTML
  • សិក្សាពីរបៀបសរសេរកំណត់ចំណាំក្នុងកូដ
  • អនុវត្តការប្រើស្លាក comment ក្នុង HTML

Step 10 ✅

ជំហានទី ១០ ✅

Use <!-- comment --> to write comments in HTML. Comments are not displayed in the browser.

ប្រើ <!-- comment --> ដើម្បីសរសេរកំណត់ចំណាំក្នុង HTML។ កំណត់ចំណាំមិនត្រូវបានបង្ហាញនៅក្នុង browser ទេ។

👉 Your Task: Type <!-- This is a comment --> below.
