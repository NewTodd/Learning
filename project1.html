Mini Project 1: Build an About Me Page

គម្រោងតូច ១៖ បង្កើតទំព័រ "អំពីខ្ញុំ"

Let’s put everything you’ve learned so far into one page! You will build a simple About Me webpage using the tags: <html>, <head>, <body>, <h1>, <p>, and <ul>.

ចូរប្រើ tag HTML ដែលអ្នកបានរៀនដើម្បីបង្កើតទំព័រ អំពីខ្ញុំ។ ប្រើ <html>, <head>, <body>, <h1>, <p>, និង <ul>។

👉 Your task: Add a title using <h1>, a short paragraph about yourself using <p>, and a list of 3 hobbies using <ul>.

👉 ភារកិច្ច៖ សរសេរចំណងជើងជាមួយ <h1> អត្ថបទខ្លីអំពីខ្លួនអ្នកដោយប្រើ <p> និងបញ្ជីចំណង់ចំណូលចិត្ត ៣ ដោយប្រើ <ul>។

🎉 Well done! You've completed your first mini project!
