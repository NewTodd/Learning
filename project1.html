📘 Project Instructions

📘 សេចក្ដីណែនាំគម្រោង

In this mini project, you'll build a simple blog layout using semantic HTML elements.

ក្នុងគម្រោងតូចនេះ អ្នកនឹងបង្កើតប្លក់មួយប្រើ Tag HTML ដែលមានអត្ថន័យ។

  • ✅ Use a <header> for the blog title
  • ✅ Add a <nav> section with links
  • ✅ Place your blog posts inside a <main> tag
  • ✅ Wrap posts with <section> and <article>
  • ✅ End with a <footer> for copyright info
  • ✅ ប្រើ <header> សម្រាប់ចំណងជើងប្លក់
  • ✅ បន្ថែម <nav> ដើម្បីចំណាត់ថ្នាក់តំណភ្ជាប់
  • ✅ ដាក់អត្ថបទក្នុង <main>
  • ✅ ប្រើ <section> និង <article> ដើម្បីរៀបចំអត្ថបទ
  • ✅ បញ្ចប់ដោយ <footer> សម្រាប់ព័ត៍មានរក្សាសិទ្ធិ

💡 Bonus Challenge: Add one more <article> with your own title and content!
