Lesson 13: HTML Input Fields

មេរៀនទី ១៣៖ បញ្ជូលព័ត៌មានក្នុង HTML

The <input> tag is used to collect user input. It can be different types such as text, email, password, and more.

<input> គឺជាធាតុសម្រាប់ឲ្យអ្នកប្រើបញ្ចូលព័ត៌មាន។ វាអាចមានប្រភេទផ្សេងៗដូចជា text, email, password និងផ្សេងទៀត។

👉 Your Task: Type <input type="text" /> in the box below to create a text input.

👉 ភារកិច្ច: វាយ <input type="text" /> ដើម្បីបង្កើតបញ្ជូលព័ត៌មាន។
