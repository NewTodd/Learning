🧠 Semantic HTML Quiz

🧠 សំណួរតេស្ត Tag មានអត្ថន័យ

1. Which tag defines the main content of the page?

១. Tag មួយណាសម្រាប់មាតិកាចម្បង?

✅ <main> is used for the main content.
✅ <main> ត្រូវបានប្រើសម្រាប់មាតិកាចម្បង។

2. What tag is typically used for navigation links?

២. Tag មួយណាសម្រាប់តំណភ្ជាប់ Navigation?

✅ <nav> is used for navigation.
✅ <nav> ត្រូវបានប្រើសម្រាប់ការរុករក (Navigation)។

3. True or False: <section> can group related content.

៣. ពិតឬមិនពិត៖ <section> អាចបែងចែកមាតិកាដែលទាក់ទងគ្នា។

✅ True. <section> is for grouping related content.
✅ ពិត។ <section> ប្រើសម្រាប់បែងចែកមាតិកាដែលទាក់ទងគ្នា។
