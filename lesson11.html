📝 HTML Basics Quiz

📝 ការសម្ភាសន៍ HTML មូលដ្ឋាន

1. What does HTML stand for?

១. HTML មានន័យថាអ្វី?

✅ HTML stands for HyperText Markup Language. It is the standard language for creating web pages.
✅ HTML មានន័យថា HyperText Markup Language។ វាជាភាសាមានស្តង់ដារ សម្រាប់បង្កើតគេហទំព័រ។

2. Which tag is used for the largest heading?

២. តើ tag ណាសម្រាប់ចំណងជើងធំបំផុត?

✅ <h1> defines the largest and most important heading on the page.
✅ <h1> គឺជា tag សម្រាប់ចំណងជើងធំបំផុត និងសំខាន់បំផុត។

3. What tag is used to add an image?

៣. តើ tag ណាដែលប្រើដើម្បីបន្ថែមរូបភាព?

✅ The <img> tag is used to embed images in HTML documents.
✅ Tag <img> ត្រូវបានប្រើដើម្បីបញ្ចូលរូបភាពនៅក្នុងឯកសារ HTML។

4. What is the purpose of the <br> tag?

៤. តើគោលបំណងរបស់ tag <br> ជាអ្វី?

✅ The <br> tag is used to insert a single line break in text.
✅ Tag <br> ត្រូវបានប្រើសម្រាប់បញ្ចូលការចេញបន្ទាត់ថ្មីមួយនៅក្នុងអត្ថបទ។
