Lesson 12: HTML Buttons

មេរៀនទី ១២៖ ប៊ូតុងក្នុង HTML

The <button> tag in HTML is used to create a clickable button on a webpage. You can add text inside the tag to label the button.

<button> ជា tag នៅក្នុង HTML ដែលប្រើសម្រាប់បង្កើតប៊ូតុងអោយអ្នកប្រើចុចបាន។ អ្នកអាចបញ្ចូលអត្ថបទនៅខាងក្នុងដើម្បីដាក់ស្លាកលើប៊ូតុង។

👉 Your Task: Type <button>Click me</button> to create a button.

👉 ភារកិច្ច: វាយ <button>Click me</button> ដើម្បីបង្កើតប៊ូតុងមួយ។
